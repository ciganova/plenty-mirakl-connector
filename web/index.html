Frequently asked

How are my API keys stored?

Encrypted at rest with per-tenant keys. EU servers (Germany). Decrypted only in-memory at sync time. We never log key material.

Can I cancel? Refunds?

Cancel any time inside the panel — billing stops at the end of the current month. 14-day trial is no-card or fully refunded if you cancel within the trial.

Which Mirakl operators are supported?

Live: Douglas, Shop-Apotheke, Otto Market. Mirakl is a standardised API — adding a new operator typically takes us 1–2 days. Just ask: contact@vagabond-consulting.com.

What happens when I hit the 200-order quota?

Sync keeps running — every order beyond 200 is billed at €0.15 and shown live in the panel. No hard stop, no failed orders, no surprise upgrade prompt.

Does it conflict with the official Plenty Mirakl plugin?

No. We are read-only on items and write only on orders + tracking. You can run the official plugin in parallel and switch over without downtime.

How fast is onboarding?

Under 10 minutes if your Mirakl key and Plenty REST user are ready. The panel walks you through it and runs a dry-import on your last 5 orders before going live.
